Device · the phone

The phone

Registers its token on every open → AppDelegate → PushBridge → device_tokens (per-device keying, prune-on-410). A tap deep-links straight to Match Detail.

Two tiers of alerts

Tier 1 — no account, scheduled locally on device: day-before reminder, Player Spotlight (iOS caps 64 pending → windowed). Tier 2 — watcher-triggered, so sign-in gated: kickoff / goals / HT / FT / lineups / V2. All opt-in; the bell tap cascades the full bundle the first time.

Which token goes where

device_tokens → V1 sends. live_activity_start_tokens → V2 push-to-start (remote-create ≤20 min pre-kickoff). Per-Activity tokens → direct updates (now largely superseded by the broadcast channel). All keyed (user_id, device_id) on a Keychain-stable device id.
